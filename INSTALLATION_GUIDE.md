# MailMind Assistant 安装与使用指南

## 📦 扩展简介

**MailMind Assistant** 是一款基于AI的智能邮件生产力工具，通过自动化邮件起草、智能优先级排序和内容摘要，显著提升职场人士的邮件处理效率。

### 核心功能
- 📝 **智能邮件起草**：根据简短指令自动生成专业邮件内容
- 📊 **邮件内容分析**：智能分析邮件优先级、情感和关键信息
- 🏷️ **智能分类与优先级**：自动为邮件分配类别和优先级
- ✨ **语言优化建议**：提供语言改进和语气调整建议

---

## 🚀 安装步骤

### 方法一：从源码安装（开发者模式）

1. **下载扩展文件**
   - 将整个项目文件夹下载到本地
   - 确保 `dist` 目录包含以下文件：
     - `manifest.json`
     - `background.js`
     - `content.js`
     - `popup.html`
     - `popup.js`
     - `icon16.png`, `icon32.png`, `icon48.png`, `icon128.png`

2. **打开Chrome扩展管理页面**
   - 在Chrome浏览器地址栏输入：`chrome://extensions/`
   - 或者点击右上角三个点 → 更多工具 → 扩展程序

3. **启用开发者模式**
   - 在扩展管理页面右上角，打开"开发者模式"开关

4. **加载扩展**
   - 点击左上角的"加载已解压的扩展程序"按钮
   - 选择项目中的 `dist` 文件夹
   - 点击"选择文件夹"

5. **验证安装**
   - 安装成功后，您会在扩展列表中看到 MailMind Assistant
   - 浏览器工具栏会显示扩展图标 🤖

## 📖 使用指南

### 1. 在Gmail中使用

#### 步骤一：打开Gmail
1. 点击扩展图标，在弹出窗口中点击"打开 Gmail"按钮
2. 或者直接访问 [Gmail](https://mail.google.com)

#### 步骤二：撰写新邮件
1. 点击Gmail页面左上角的"撰写"按钮
2. 等待几秒钟，编辑器右上角会出现 **🤖 智能助手** 按钮

#### 步骤三：使用AI功能
点击"🤖 智能助手"按钮，会弹出功能菜单，包含四个选项：

##### 📝 智能起草
- **用途**：根据简短指令自动生成完整邮件
- **使用方法**：
  1. 点击"智能起草"
  2. 在弹出对话框中输入指令，例如：
     - "邀请团队参加周三的项目会议"
     - "感谢客户的反馈并说明解决方案"
  3. 系统会自动生成邮件内容并填充到编辑器

##### 📊 邮件分析
- **用途**：分析邮件内容的类别、情感和优先级
- **使用方法**：
  1. 在编辑器中输入或粘贴邮件内容
  2. 点击"邮件分析"
  3. 查看弹出窗口中的分析结果，包括：
     - 邮件类别
     - 情感倾向
     - 优先级等级和原因
     - 建议的行动项
     - 相关标签

##### 📄 生成摘要
- **用途**：为长邮件生成简短摘要
- **使用方法**：
  1. 在编辑器中输入或粘贴较长的邮件内容
  2. 点击"生成摘要"
  3. 系统会显示简洁的摘要文本

##### ✨ 语言优化
- **用途**：优化邮件的语言表达和专业程度
- **使用方法**：
  1. 在编辑器中输入邮件草稿
  2. 点击"语言优化"
  3. 系统会自动优化语言并替换原内容

### 2. 在Outlook中使用

使用方法与Gmail类似：

1. 访问 [Outlook](https://outlook.live.com) 或 [Outlook Office](https://outlook.office.com)
2. 点击"新建邮件"开始撰写
3. 等待"🤖 智能助手"按钮出现
4. 使用相同的功能菜单操作

---

### 3. 右键菜单快捷功能

扩展还提供了右键菜单快捷方式：

1. **在任何网页上选中文本**
2. **右键点击选中的文本**
3. **选择以下选项之一**：
   - "使用MailMind分析" - 快速分析选中的邮件内容
   - "使用MailMind生成摘要" - 快速生成选中内容的摘要

## 🔧 故障排除

### 问题1：看不到"🤖 智能助手"按钮

**解决方法**：
- 确保您已经点击了"撰写"或"新建邮件"按钮
- 刷新Gmail/Outlook页面
- 检查扩展是否已启用（在 chrome://extensions/ 中查看）
- 等待几秒钟让扩展完成初始化

### 问题2：功能无法使用

**解决方法**：
- 按F12打开开发者工具，查看Console中的错误信息
- 在 chrome://extensions/ 中重新加载扩展
- 清除浏览器缓存后重试

### 问题3：内容没有填充到编辑器

**解决方法**：
- 确保光标在编辑器内
- 检查编辑器是否处于激活状态
- 手动点击编辑器区域后再次尝试

### 问题4：CSP安全策略错误

**解决方法**：
- 确保使用最新版本的扩展文件
- 检查 popup.html 中没有内联JavaScript
- 重新加载扩展

## 🛠️ 开发者信息

### 技术栈
- Chrome Extension Manifest V3
- Vanilla JavaScript
- Chrome Storage API
- Chrome Tabs API
- Chrome ContextMenus API

### 项目结构
```
dist/
├── manifest.json       # 扩展配置文件
├── background.js       # 后台服务脚本
├── content.js          # 内容脚本（页面注入）
├── popup.html          # 弹出窗口页面
├── popup.js            # 弹出窗口脚本
└── icon*.png           # 扩展图标
```

### 权限说明
- **storage**: 保存用户偏好设置
- **activeTab**: 访问当前活动标签页
- **scripting**: 注入内容脚本
- **contextMenus**: 创建右键菜单
