# MailMind Assistant 图标

本目录包含 MailMind Assistant Chrome扩展的图标文件。

## 图标规格

需要以下尺寸的PNG图标：
- icon16.png (16x16) - 扩展栏小图标
- icon32.png (32x32) - Windows系统图标
- icon48.png (48x48) - 扩展管理页面图标  
- icon128.png (128x128) - Chrome应用商店图标

## 设计建议

图标应该：
- 使用蓝色主色调 (#1890ff)
- 包含邮件或AI相关的视觉元素
- 在小尺寸下仍清晰可见
- 与应用的整体设计风格保持一致

## 当前状态

目前使用占位符图标。建议后续使用专业设计工具创建正式图标。
